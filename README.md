# Spring-Image-Uploader-MVC

This is a small software creation made for uploading images using Spring MVC

###  `Front End`
* HTML
* Javascript
* CSS
* Boostrap Framework
* AJAX

###  `Back End`
* Spring
* Maven
* Hibernate
* Lombok
